# Milestones & Performance Targets

This document sets the numeric benchmarks, compounding schedules, and performance targets required to measure the bot's success.

---

## 📈 Milestones & Target Schedule

| Period | Metric | Target Metric | Category | Description |
| :--- | :--- | :--- | :--- | :--- |
| **Week 1–2** | Environment Setup | API connectivity | Technical | Connect bot to Binance API, fetch live ticker data with 0 errors. |
| **Week 3–4** | Strategy Activation | Signal generation | Technical | Run all 3 strategies in sandbox/paper mode generating clean signals. |
| **Week 5** | Simulation Performance | Win Rate > 50% | Performance | Maintain greater than 50% win rate across all strategies. |
| **Week 6** | Strategy Promotion | Win Rate > 55% | Performance | Promoted strategy must achieve >55% win rate on Binance Testnet. |
| **Week 6** | Profitability Floor | Avg Profit > 0.4% | Performance | Net average profit per trade must exceed 0.4% after all fees. |
| **Week 6** | Drawdown Limit | Max Drawdown < 8% | Risk | Under paper trading, max drawdown should not exceed 8% of ₹1,000. |
| **Week 7** | Dashboard Launch | Telemetry working | Technical | Streamlit dashboard fully active, displaying SQLite trade logs. |
| **Week 8** | Production Launch | Live execution | Milestone | First live-money trade completed with real stop-loss verified. |
| **Month 2** | Live Performance | Win Rate > 55% | Performance | Maintain >55% win rate over 4 consecutive weeks of live trading. |
| **Month 2** | Live ROI | Monthly return > 15% | Financial | Reach ₹1,150+ ending capital from the initial ₹1,000 weekly budget. |
| **Month 3** | Capital Scaling | Increase weekly budget | Financial | Increase fund to ₹2,000/week if all Month 2 targets are met. |
| **Month 6** | Compounding Goal | Compounded Capital | Financial | Grow initial ₹1,000 fund to ₹2,500 – ₹3,500. |

---

## 💰 Compounding Capital Projection

These figures represent a realistic **2% – 3% weekly compounding** schedule. This model assumes disciplined risk management and adherence to the promoted strategy.

```
[Start]   Week 1:  ₹1,000
[Phase 1] Month 1: ₹1,150  (+15%)
[Phase 2] Month 2: ₹1,320  (+32%)
[Phase 3] Month 3: ₹1,520  (+52%)
[Phase 4] Month 6: ₹2,800  (+180%)
[Phase 5] Year 1:  ₹7,800+ (+680%)
```

> [!WARNING]
> These projections are based on statistical simulation and historical averages. Live market volatility, slippage, and execution anomalies can impact performance. Paper trading is mandatory to establish empirical base rates.
