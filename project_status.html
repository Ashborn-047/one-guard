⚠️ Core Implementation Hurdles

⏰ Clock Synchronization Offset
Windows and VPS hosts regularly drift. We test offset dynamically against Binance servers (must be < 1000ms offset) to avoid request authentication failures.
🚦 CCXT Rate Limiters
Binance enforces a weight-limit (typically 1200/min). We configure `enableRateLimit: true` and cache calculations locally in SQLite to restrict exchange hits.
🔒 Key Safety Policy
API credentials must NEVER have withdrawal permissions active. Live trades must always be preceded by Sandbox Testnet validation.
